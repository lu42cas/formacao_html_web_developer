Formaçação HTML Web Developer

No bootcamp de Formação HTML Web Developer você será introduzido ao desenvolvimento web, aprendendo sobre o HTML, algumas de suas tags e atributos. Para essa formação, você precisará de um navegador, um editor de códigos* e acesso à internet.

*Aqui utilizaremos o editor de código chamado VisualStudio Code.

Veja alguns exemplos de tags aprendidas

  • p: demarca o início e fim de parágrafos na sua página
  • h1: utiliza uma fonta maior, geralmente aplicada ao nome do site
  • ol: tag que permite a criação de uma lista ordenada
  • ul: cria uma lista não ordenada
  • li:utilizada para listar os itens dentro de uma lista ordenada/ou não ordenada
  • blockquote:para centralizar o texto como uma citação
  • mark: para destacar o texto
  • a: para criar um link
  • i: para deixar os caracteres em itálico
  • del: para identificar que parte do texto foi deletada
